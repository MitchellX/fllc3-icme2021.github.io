The 3rd Grand Challenge of 106-Point Facial Landmark Localization

ICME 2021

July 5 2021, Shenzhen, China

Dataset Description




    The structure of files that competitors can get is shown below. The file picture_mask means face photos with the virtual or real mask. The algorithm is publicly opened in FaceX-zoo[3]. Besides, we use Lapa[2], a face parsing module, to deal with facial occlusion. We highly encourage competitors to use their own mask-add algorithms to add virtual masks, so we provide the original non-masked pictures, called picture.

Figure1: Structure of file


Training dataset:


We collect an incremental dataset named JD-landmark-mask. Besides the face images in JD-landmark[1] dataset, we provide the virtual-masked face images by utilizing our virtual mask addition algorithm[3] This dataset, containing about 40,000 faces, is accessible to the participants (with landmark annotations). participants should train their model on the masked pictures. The images and landmarks will be released on March 1, 2021.
Fig.2 shows some examples in which the first row refers to the real face image and the second row refers to the virtual masked image.

Figure2: Examples of training dataset. The first row: non-masked face image. The second row: virtual-masked image

Validation dataset:


It consists of 2000 virtual-masked images which are generated by the same method as the training dataset. The participantsâ€™ models will be evaluated on this set before the final evaluation. The masked images without ground truth will be released on March 1, 2021, simultaneously.

Test dataset:


It contains 2000 in-the-wild real-masked images and 2000 virtual-masked images which are collected from the internet. It will be used for the final evaluation.
To prevent cheating on the test set, the test images will not be released to participants. And participants need to submit their model within one week, from April 1 - April 7.

    Note!:   the test dataset is blind to participants throughout the whole competition.
